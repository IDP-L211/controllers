execute_action(actions: list) → bool¶

Execute the first action in a set of actions

When each action is completed it’s removed from the list. Using list mutability this allows us to alter / check the action list elsewhere in the code to see the bots progress and also change its objectives. If people have better ideas on how to do this I’m all ears.

Actions:

  • move: Go to the given coordinates

  • turn: Face the given bearing

Parameters

actions (list) – Each list element is a tuple/list of [“action_type”, value]

Returns

Whether action list is completed or not

Return type

bool
